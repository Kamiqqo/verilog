В файле верхнего уровня закомментированны входные и выходные сигналы, не взаимодействующи напрямую с платой, для генерации файла с расширением .bit
файл CORDIC реализует трансцендентную функцию, необходимую для вычисления ряда Фурье принимающего сиусы и косинусы.
файл SevenSegmentLED реализует простую логику вывода числовых данных через семисегментные индикаторы в данной работе на плаte digilent nexys a7
файл TOP_FURRY является файлом верхнего уровня проекта, который попутн реализует логику дешифрации PDM сигнала поступающего с микрофона платы а также генерирует псевдослучайные числа основываясь на амплитудах, получаемых  ходе работы модуля furry 
файл atan_table содержит таблицу углов подающихся в CORDIC
файл clk_div реализует делитель частоты
файл constr это проектные ограничения для отладочной платы
файл count это счетчик
файл furry реализует логику Ряда Фурье для анализа спектра звукового сигнала методом Дискретного Преобразования Фурье(ДПФ) и соответвтвенного высчитывания амплитуд звуковых сигналов
файл test_gen это тестовый модуль модуля верхнего уровня
файл test_furrie это тестовый мдуль модуля реализации ДПФ
файл filtercon - фильтр дребезга контактов дл возможног подключения внопок на плату
файл dtrigger реализует простейший Д-триггер
