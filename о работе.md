В ветке представленна реализация гнератора случайных чисел на основе шумов считываемых микрофоном отладочной платы atrix a7 100t
модуль CORDIC реализует трансцендентную функцию, которая необходима для высчитывания формулы фурье в модуле furry
модль count реализует обычный счетчик, модуль dtrigger реализует модуль D триггера
модуль filtercon реализует логику фильтра дребезга контактов для реализации нажатия на кнопку на плате
модуль furry реализует спектральное разложение звукового сигнала методом Дискретного Пребразования Фурье
В SevenSegmentLed расписана логика работы семисегментных индикаторов для вывода генерируемых чисел
модуль synchro обычный синхронизатор
wergsg - это модуль верхнегоровня реализующий логику считывания и дешифрации дискретизированного сигнала в виде PDM с микрофона на плате и передачи его в модуль furry где реализуется последующая логика
